<h2>功能介紹</h2>

- 提供使用者註冊及登入
- 登入者可以將喜愛的電影加入我的最愛
- 提供使用者搜尋電影
- 串接電影 API 並回傳時下熱門電影
- 點選電影圖示彈出詳細訊息，包含評價、概述等等 ...

<h2>使用技術</h2>

- SCSS
- JavaScript
- React.js
- Fetch
- Axios
- Node.js
- Express.js
- MongoDB
